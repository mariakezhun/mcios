# МЦИОЗ

Репозиторий сайта научного центра МЦИОЗ. 

## Функционал

Репозиторий является хранилищем кода, который в последствии деплоится на тильду с спользованием блоков "html"

## Используемые технологии

- HTML5
- CSS:
    - Flexbox
    - Grid Layout
    - Media Queries
- Методология БЭМ
- Фйловая структура Nested БЭМ
- JavaScript
